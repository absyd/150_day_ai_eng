## 🏗️ Curriculum Structure

### Phase 1 — Foundations (Days 1–35)
- **Skills**: LLM APIs, Prompt Engineering, Embeddings, Vector DBs, RAG (4 variants), Production SaaS, Auth, Billing, CI/CD, Observability
- **Milestones**: Chat with PDF SaaS + Production RAG SaaS

### Phase 2 — Agents & Orchestration (Days 36–65)
- **Skills**: Agentic loops, LangGraph, Claude Agent SDK, Hooks, Multi-agent, MCP (custom + community), Voice AI, Vision AI
- **Milestones**: Multi-agent research system + Voice assistant + Custom MCP server

### Phase 3 — Production Systems (Days 66–99)
- **Skills**: Data AI, NL→SQL, LLMOps, Cost/latency optimisation, Security, Prompt injection, Structured output mastery, Batch processing
- **Milestones**: Data AI platform + Structured extraction pipeline

### ML Depth — Parallel Track (Days 1–30)
- **Skills**: NumPy, Pandas, scikit-learn, PyTorch from scratch, CNNs, LSTMs, Transformer attention, Keras, HuggingFace, LoRA/QLoRA, MLOps
- **Milestones**: Fine-tuned Llama + Dockerised model endpoint

### Phase 4 — Deep ML (Days 100–120)
- **Skills**: Classical ML benchmarks, PyTorch Trainer, ResNet fine-tuning, BERT fine-tuning, QLoRA, RAG vs fine-tune, MLflow, Evidently drift
- **Milestones**: ML model serving pipeline + Drift monitoring system

### Phase 5 — Pro Tier (Days 121–150)
- **Skills**: System design at scale, Claude Code mastery, Capstone project, Portfolio, Resume, Blog, Certification, Interview mastery
- **Milestones**: Capstone (deployed, case study published) + Certified + Offers
